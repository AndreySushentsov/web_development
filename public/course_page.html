О чем это

Изучите Front-end разработку и разработку гибридных мобильных приложений, построите серверную часть и реализуете полностью функциональное приложение.

Первые три части в включают в себя разработку на стороне клиента, охватывают HTML / CSS, JavaScript / JQuery и работу с такими фреймворками как AngularJS и Bootstrap. На стороне сервера, вы научитесь реализовывать базы данных NoSQL с использованием MongoDB, работать в среде Node.js и "общаться" со стороной клиента через RESTful API. Кроме того, вы научитесь создавать гибридные мобильные приложения, используя структуру Ionic и Cordova.Вы будете применять свои навыки, чтобы построить полнофункциональное веб-и гибридное мобильное приложение с серверной поддержкой.

Разделы

1-ая часть

HTML, CSS, JavaScript

Подробнее

Этот курс научит вас основам разработки веб-страниц, он охватывает HTML, CSS и JavaScript. От вас не потребуется знаний этих технологий, будет плюсом если у вас есть некоторый опыт программирования. Первыми будут разбираться,HTML и CSS. Затем мы перейдем к рассмотрению основных компонентов JavaScript, включая переменные, массивы, циклы, события и функции. Затем мы разберем более продвинутые элементы управления JavaScript, включая расширенные возможности использования функций, контроль событий, обработку массивов и манипулирования DOM.


После завершения этой части, вы сможете:

- Создавать веб-страницы с помощью HTML-элементов
- Сможете применять CSS на части веб-страницы, для изменения отображения и поведения.
- Создавать интерактивные веб-страницы с помощью JavaScript
